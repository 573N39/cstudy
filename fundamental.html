Fundamentals of C

The basic code names and the functions of C; functions for basic input/output, easy expressions and the words that are used in C.

Basic structure

To give a simple output of literal string, we use the function printf(). Here we have the basic structure of a C program.

The code has 'directive; standard input/output(#include)', and the function int main(void), then the statement printf() where printf() is a function too but placed as a statment; a function-call. printf("Hello World") can show the literalstring Hello World on the console screen. A crucial part to know is that if it's just written printf("Hello World");, then the next words are continued on the same line instead of starting from the next, so we have to use \n to continue the next words from next line. For example, printf("Hello World\n");. Also, in statements we have to end most of the functions or expressiosn with a semicolon because it means a fullstop of a sentence to them.

As shown in the above image, \n had used in printf() function and before seeing how it would be shown on the console, let's check out what it means for the blue words at the top of the code. Basically they are Comments that the programmers use not to make an output on the console but to make some notes within the codes for the understanding sakes. There are 2 types; // and /**/. If we write /*Hello*/ then the word Hello becomes just a comment that effects nothing to the code. It doesn't matter of the lines and spaces but the comment must begin with /* and closed by */ or else if we don't close the comment, C understands everything after /* as comments. // is a new tech of C99 that makes an end of a comment automatically after a line's written. // is good for a short one-lined comment.

Constant and Variable

  • A constant means a value that can't be changed, like if we write printf("7 is the lucky number") then it doesn't change unless the programer changes the data.
  • A variable simply means a storage of data but it's called as a variable because unlike the constants, the values can be easily changed. As told a storage, whatever the values in the storage can be replaced by others at anytime.

    • To use any variable, we must understand the 3 things; types, declaration and assignment. There are basically 3 types of string; literal, integer and floating string but in this chapter we're going to deal with only numerical strings except literals. To make a variable of integer values, we need int and float for floating values; decimals.

    Declaration and Assignment

    Declaration is the process to declare a variable that which type it belongs to. For example int age; declares the variable named 'age' that would contain an integer value and float weight; declares the variable named 'weight' with a decimal values.
    Assignment is the process to assign any value into a declared variable.
    int age;
    float weight;
    age = 33;
    weight = 65.5f;
    As shown above, firstly the variables must be declared and then assign any values.
    There are some important points to be considered before shifting to next studies;

    • Till C89, the declarations had to be written before any kind of other statements but after C99, it doesn't really matter any more but of course, the assignment can't be progressed before the declaration as the variables must be defined first.
    • A floating value should better contain f at the end of the decimal value to avoid any kind of error.
    • Declaration and Assignment could be done together on the same line;
      int age = 33;
      float weight = 165.5f;
    • While we assign a value to a variable, inputing the value is called as intialization and the value is called an intitializer.

      • Printing process

      Let's use the same variables that were used above; age & weight. Before printing any value, just remember the 2 things; %d and %f.

      For an integer variable; age, the code must be like below,
      printf("age: %d\n", age);
      age: is just a literal string and %d is a keyword that calls integer value of age; 33.

      For a floating variable; weight, it's like below,
      printf("weight: %f\n", weight);
      weight: is just a literal string as well and %f is a keyword that calls floating value of weight; 165.5.

      Basic expression

      Not only a single value but it's possible to assign an expressions to a variable; basic calculations.
      int cal1 = 1 * 2 + 4 - 2;
      float cal2 = 50 / 2:
      The computer calcuates these expressions and saves the values into the variables. If these values were printed then the console screen will display the result of calculations instead of the expressions themselves.

      Reading input

      We had seen how to print any value on the screen but then now it's time to input a value on the screen directly instead of writing any value within the code.
      To do this, we use the function scanf() and a variable for it.
      int age = 33;
      float weight = 165.5f;
      printf("Enter your age: ");
      scanf("%d", &age);
      printg("Enter your weight: ");
      scanf("%f", &weight);
      Like the same way we did in printf of a variable, even in scanf we put %d or %f according to the type and put the name of the variable on the right side but must be careful not to miss "&" sign in front of the name.
